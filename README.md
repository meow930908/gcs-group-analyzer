# gcs-group-analyzer

網址：https://meow930908.github.io/gcs-group-analyzer/

傳說對決 GCS 例行賽排名分析程式 - By 小律（2025/11/21)
能完整分析未來所有比賽結果
1. 大分較多的隊伍 排名較前
2. 若大分相同，勝差（小分 - 敗場）較多的隊伍 排名較前
3. 弱勝差相同，彼此對戰結果之勝差較多的隊伍 排名較前（目前設定兩隻隊伍，三隻以上設定過於麻煩，自行判斷）

輸入格式如下（詳情請看網站）：

若雙方已對戰：隊伍A 比分 比分 隊伍 隊伍B （範例：FW 3 1 ANK）

若雙方未對戰：隊伍A 0 0 隊伍B（範例：FW 0 0 ANK）
